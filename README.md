# Tugas Praktikum Pertemuan 11 Pemrograman Web - Operator JavaScript
Repositori ini berisi kode implementasi materi dasar JavaScript mengenai penggunaan berbagai jenis operator. Terdapat 3 file HTML utama yang mensimulasikan studi kasus di dunia nyata (seperti pengelolaan data siswa, kasir toko, dan perhitungan belanja komputer).

## 📂 Struktur File
1. **`Operator_Aritmatika.html`**: Implementasi perhitungan matematika dasar (Penjumlahan `+` dan Pengurangan `-`) pada studi kasus transaksi Toko Komputer.
2. **`Operator_Penugasan.html`**: Implementasi operator penugasan (`+=` dan `-=`) untuk memperbarui stok barang dan kas keuangan toko secara dinamis.
3. **`Operator Pembanding.html`**: Implementasi logika perbandingan (`>=`, `>`, `<`, `<=`, `==`, `!=`) untuk menentukan status kelulusan siswa berdasarkan KKM.
